Features of HTML5

Features of HTML5 are as follows :-

Form Validation

HTML5 offers built-in form validation, reducing the need for custom JavaScript code.



Video playback:

The < video> HTML element embeds a media player which supports video playback into the document.

Semantic Tags:

A semantic element clearly describes its meaning to both the browser and the developer.

Article Heading

The < article> tag specifies independent, self-contained content.

iFrame tag:

The iframe in HTML stands for Inline Frame. The ” iframe ” tag defines a rectangular region within the document in which the browser can display a separate document, including scrollbars and borders.
